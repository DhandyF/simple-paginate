Vanilla JS Pagination Plugin

Lightweight, dependency-free pagination with smart ellipsis, icon navigation, size presets, search integration, and fully customizable colors.

Live Demo

Check the live preview: Demo Site

Overview

This plugin makes it easy to paginate lists, tables, or any dataset in plain JavaScript — no dependencies. It supports smart ellipsis for compact navigation, previous/next & first/last icons, multiple size modes, search/filter compatibility, and full style customization.

Features

  • Dependency-free — pure JavaScript
  • Customizable active/hover colors
  • Smart ellipsis navigation
  • Supports small, medium, and large sizes
  • Works with lists, tables, or custom renderers
  • Callback on page change
  • Lightweight (minified & gzipped)

Installation

<link rel="stylesheet" href="style.min.css">
<script src="pagination.min.js"></script>

Basic Usage

<div id="list"></div>
<div id="pagination-controls"></div>

<script>
const data = Array.from({length:42},(_,i)=>`Item ${i+1}`);
const paginator = new Pagination(document.getElementById('list'),{
  data,
  itemsPerPage:10,
  maxVisiblePages:5,
  paginationId:'#pagination-controls',
  size:'medium',
  textColor:'inherit',
  bgColor:'transparent',
  activeBgColor:'#2563eb',
  activeTextColor:'#ffffff',
  hoverBgColor:'#1f2937',
  hoverTextColor:'#e5e7eb',
  onPageChange:function(page){console.log('Page',page);}
});
</script>

Options

NameTypeDefaultDescription
dataArray[]Items to paginate
itemsPerPageNumber10Items per page
maxVisiblePagesNumber5Max numbered buttons before ellipsis
paginationIdString-CSS selector for the controls container
sizeStringmediumsmall | medium | large
textColorStringinheritInactive text color
bgColorStringtransparentInactive background color
activeBgColorString#007bffActive button background
activeTextColorString#ffffffActive button text color
hoverBgColorString#e0e0e0Hover background (inactive)
hoverTextColorString#000000Hover text color (inactive)
onPageChangeFunction()=>{}Called on page change; receives current page

API

  • paginate() — returns current page slice of data
  • goToPage(n) — jumps to page n
  • render() — re-renders items & controls
